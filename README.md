Clash Royale Card Finder

This is a simple React app where you can search for Clash Royale cards and see their details like stats and images.
I built this to learn React, Tailwind CSS, and how to fetch data from an API.


Features:
Search for cards by name

View card stats and images

Simple and clean design with Tailwind CSS

p.s(This is my first React project and I’m still learning. The app uses a Clash Royale API to get card data.)
